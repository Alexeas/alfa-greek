Зависимое наклонение глаголов в греческом языке

Выражает:
  1. Желание:
    Όταν έρθουν τα Χριστούγεννα, θα φύγουμε στην Αθήνα. – Когда придёт Рождество, мы уедем в Афины.
  2. Приказ (просьбу):
    Μην του επιτρέπεις να βγαίνει χωρίς τα ζεστά ρούχα το χειμώνα έξω. – Не разрешай ему выходить без тёплой одежды зимой на улицу.
  3. Сомнение:
    Τι να κάνω; Να αγοράσω ή να μην αγοράσω αυτό το αυτοκίνητο. – Как мне быть? Покупать или не покупать этот автомобиль?
  4. Пожелание (с частицей ας – пусть):
    Ας περάσουμε καλά στις διακοπές! – Хорошо нам отдохнуть на каникулах!
  5. Возможность:
    Αν τον βρω θα του μιλήσω. – Если я его найду, я с ним поговорю.
  6. Разрешение:
    - Μαμά, να παω και εγώ εκδρομή με άλλα παιδιά; - Να πας. – Мама (можно) я поеду с остальными ребятами на экскурсию? - Поезжай .
Образуется: при помощи таких частиц и союзов, как ας, να, αν, εάν, για να, όταν, άμα, μήπως, πριν (να), μη(ν).

I. Зависимое наклонение несовершенного вида (Συνεχής Υποτακτική)

Выражает: многократное повторяющееся действие.
Образуется: Να + ρήμα σε Ενεστώτα (глагол в настоящем времени).
Таблица спряжения глаголов в зависимом наклонении несовершенного вида:
Местоимение Акт. Залог Пас. Залог
Εγώ (Я) Να χτίζω Να χτίζομαι
Εσύ (Ты) Να χτίζεις Να χτίζεσαι
Αυτός / Αυτή / Αυτό (Он, Она, Оно) Να χτίζει Να χτίζεται
Εμείς (Мы) Να χτίζουμε Να χτίζόμαστε
Εσείς (Вы) Να χτίζετε Να χτίζεστε
Αυτοί / Αυτές / Αυτά (Они) Να χτίζουν Να χτίζονται

II. Зависимое наклонение совершенного вида (Απλή Υποτακτική)

Выражает: однократное вероятное действие в будущем.
Образуется: Να + θέμα Αορίστου + καταλήξεις ενεστώτα (основа Аориста и окончание глаголов в настоящем времени).
Таблица спряжения глаголов в зависимом наклонении совершенного вида:
Местоимение Акт. Залог Пас. Залог
Εγώ (Я) Να χτίσω Να χτιστώ
Εσύ (Ты) Να χτίσεις Να χτιστείς
Αυτός / Αυτή / Αυτό (Он, Она, Оно) Να χτίσζει Να χτιστεί
Εμείς (Мы) Να χτίσουμε Να χτιστούμε
Εσείς (Вы) Να χτίσετε Να χτιστείτε
Αυτοί / Αυτές / Αυτά (Они) Να χτίσουν (-ουνε) Να χτιστούν (-ούνε)

III. Зависимое наклонение глаголов перфектной формы (Υποτακτική Παρακειμένου)

Выражает: действие, предшедствующее другому действию.
Образуется: Να + ρήμα σε Παρακείμενο (глагол в сложном прошедшем времени).
